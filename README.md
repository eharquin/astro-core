# astro-core

Astro-core is a modular C++ rendering engine core with a Vulkan backend.
It provides:
- An application & layer system
- A window and input abstraction (GLFW)
- A rendering API abstraction
- A Vulkan implementation of the rendering backend

This repository contains:
- `core/`   → reusable engine core
- `app/`    → example / test application
